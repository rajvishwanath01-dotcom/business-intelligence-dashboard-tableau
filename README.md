# business-intelligence-dashboard-tableau
Retail sales performance dashboard built in Tableau with KPIs, filters, and business insights for data-driven decision making.
**Project Overview
**This project presents an interactive dashboard built using Tableau to analyze retail sales performance across different regions, product categories, and time periods. The dashboard helps identify trends, monitor key metrics, and support data-driven business decisions.

Objectives
Analyze overall sales, profit, and order performance
Identify top-performing products and categories
Understand regional sales distribution
Track monthly sales trends

Dashboard Features
KPI cards for Total Sales, Profit, and Orders
Interactive filters (Date, Region, Category)
Monthly sales trend analysis
Top 5 products by revenue
Profit contribution by category
Regional sales comparison
“Business Insights” section for quick understanding

Key Insights
Technology category generates the highest profit
East region leads overall sales performance
Sales peaked during March
Office Supplies shows the lowest profitability


Design Approach
Used blue for sales and green for profit to maintain intuitive understanding
Applied consistent spacing and alignment for a clean layout
Highlighted top-performing areas for better storytelling

Tools Used
Tableau (Data Visualization)
Excel / CSV (Data Source)

Files Included
Retail_Sales_Dashboard.twbx → Tableau dashboard file
dashboard_preview.png → Dashboard screenshot
README.md → Project documentation

How to Use
Download the .twbx file
Open it in Tableau Public/Desktop
Use filters to explore the dashboard interactively
